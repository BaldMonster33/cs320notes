Notes for git

Commiting

git commit: make a new commit.(No name)

Branching

git branch NAME: make a new branch called NAME

git checkout NAME: put us on a new branch

git checkout -b NAME: put us on a new branch named NAME

Moving around

git merge NAME :merge the branch NAME into HEAD. This would create a new commit one.

git rebase NAME: copy from HEAD to under NAME. If HEAD is an ancestor of NAME, HEAD will be moved forward.

relative refs: Moving up with "^", and ~<num> for a number of times. Using with git checkout NAME

git branch -f NAME HEAD~nums: move NAME to places ahead of HEAD for nums

git reset HEAD~nums: revert changes by moving backwards.(local)

git revert HEAD: reverse changes and share with others.(Make a new ')

Local and Cloud

git init; git add FILE; git commit -m "NOTES": initialization, adding file and memo.

git remote add NAME URL; git push -u NAME BRANCH : submit to Github
